# Reverie Eval Snapshots

Sample JSON artifacts generated by `npx tsx sdk/native/examples/reverie/eval/snapshot.ts`.

- `sample-slices.json` – minimal demo slices bundled with the repo so CI/docs have deterministic data.
- `slices.json` / `user_messages.json` / `assistant_responses.json` – automatically overwritten when you run the snapshot helper on your own `~/.codex/sessions` directory.

Use these files with `eval/index.ts` via `--dataset` to benchmark strategies without hitting live transcripts.
