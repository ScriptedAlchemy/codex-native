# Changelog

All notable changes to this project will be documented in this file.

## 0.0.1 - 2025-11-11

- Added `codex-native` CLI with `run` and `tui` subcommands, including automatic config
  discovery, plugin loading, and approval/tool hooks.
- Exposed TUI bindings and headless test helpers to keep parity with the Rust TUI build.
- Documented CLI usage, configuration files, plugin development, and release procedures.
- Added smoke tests for the CLI and streaming tool interceptors.
